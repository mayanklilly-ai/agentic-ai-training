# Agentic AI Training Notebook

Hands-on training session covering Tokenisation, Prompt Engineering,
RAG Pipelines, AI Agents with LangGraph, MCP, and A2A.

## 🚀 Launch the Training Environment

(https://github.com/mayanklilly-ai/agentic-ai-training)

Click the button above. Your environment will be ready in ~2 minutes.
No installation needed. The notebook opens automatically.

## What is included

- training_notebook.ipynb — the full 7-section hands-on notebook
- .env.example — shows the environment variable needed (HF_TOKEN)

## API Key

The HF_TOKEN is pre-loaded into the environment automatically.
You do not need to create a .env file when using Codespaces.

If running locally, copy .env.example to .env and fill in your token.
Get a free token at: https://huggingface.co/settings/tokens
